Regex reference (Python re)

Applies to Regex match and Regex replacement.

Quick patterns

GoalPatternNotes
Digits\d+Same as [0-9]+
Word characters\w+Letters, digits, underscore
Whitespace\s+Spaces, tabs, newlines
Anything.*Greedy; use .*? for non-greedy
File extension\.([A-Za-z0-9]+)$Captures extension without dot

Anchors

  • ^ start, $ end
  • \A start (strict), \Z end (strict)
  • \b word boundary, \B not a boundary

Character classes

  • [abc] any of a/b/c
  • [^abc] anything except a/b/c
  • [A-Z] range
  • . any char except newline (unless DOTALL)

Groups & alternation

  • (...) capturing group
  • (?:...) non-capturing
  • (?P<name>...) named capturing group
  • a|b alternation

Quantifiers

SyntaxMeaning
?0 or 1
*0 or more
+1 or more
{m}exactly m
{m,n}between m and n
*?, +?, {m,n}?non-greedy (lazy)

Replacement references

  • \1, \2, ... numeric group reference
  • \g<name> named group reference

Example

Example match:       ^(?P<year>\d{4})_(?P<title>.+)$
Example replacement: \g<title> (\g<year>)
Example result:      2026_My Movie.mp4 → My Movie (2023).mp4
