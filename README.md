# Dion

This repo hosts the flatpak version of [Dion video conferencing service with a hybrid cloud infrastructure.](https://diongo.ru/)

Note that this is an **unofficial** redistribution.

## Issue reporting

**Please only report issues in this repo that are specific to the flatpak version.**

Issues that can be replicated in another release should be reported via mailto:support@diongo.ru.
Make sure, that the reported issue is **not** flatpak-related.
