# CMCC Cloud PC

CMCC Cloud PC (Chinese: 移动云电脑)is a cloud desktop client software provided by China Mobile, supporting remote access to cloud virtual desktops.

## About This Repository

The CMCC Cloud PC application itself is **proprietary software** (closed source).

This Flatpak package is **not** verified, affiliated with, or supported by China Mobile.

## Build Instructions

This Flatpak package is built based on the [official CMCC Cloud PC UOS (Tongxin) DEB package](https://soho.komect.com/clientDownload).

### Architecture Support

- x86_64 (AMD64)
- aarch64 (ARM64) - Not verified. If you can successfully install and use it on ARM devices, please submit a PR to remove this note.
