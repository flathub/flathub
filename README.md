# uk.org._6809.xroar
flatpak build of XRoar, a Dragon and Tandy 8-bit computer emulator.

**NOTE:** This package does not come with the roms necessary to run the emulated machines. It is up to the end user to source the rom images.

## XRoar

For more information on how to use the emulator please see the XRoar project website: https://www.6809.org.uk/xroar/.

The desktop icon was sourced from the XRoar source code at `src/windows32/xroar-256x256.ico` and converted to a PNG image.
