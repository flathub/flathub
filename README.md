Flatpak Manifest for OMERewriter
================================

This repository contains the Flatpak manifest for OMERewriter, a small tool to
view and rewrite OME-TIFF metadata, for deployment on Flathub.

To learn more about OMERewriter, please visit the project's website at
[github.com/bothlab/omerewriter](https://github.com/bothlab/omerewriter).
