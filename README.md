# Actual

Actual Budget is a super fast and privacy-focused app for managing your finances. At its heart is the well proven and much loved Envelope Budgeting methodology.

This repo hosts the flatpak wrapper for [Actual](https://actualbudget.org), available at [Flathub](https://flathub.org/apps/details/org.actualbudget.actual).

```sh
flatpak install flathub org.actualbudget.actual
flatpak run org.actualbudget.actual
```
