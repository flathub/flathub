# Helden-Software

Ein Generator für "Das schwarze Auge" (DSA 4.1).
[https://helden-software.de](https://helden-software.de)

**This app is only available in German, because it's based on german-only rule books.**


## Flatpak Development
Use `make` to test your changes before creating a PR:
- `make install` - build and install this flatpak in local repository
- `make run` - build, install, and run this flatpak
- `make validate` - checks that all manifests/metainfo/desktop files are valid
- `make dist` - builds bundle for publication
