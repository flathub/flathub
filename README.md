# OrcaSlicer Flatpak

Official [Flathub](https://flathub.org/) build for [OrcaSlicer](https://github.com/OrcaSlicer/OrcaSlicer) — a popular open-source 3D printing slicer with support for Bambu, Prusa, Voron, VzBot, RatRig, Creality, and more.

## Install from Flathub

```bash
flatpak install flathub com.orcaslicer.OrcaSlicer
flatpak run com.orcaslicer.OrcaSlicer
```
