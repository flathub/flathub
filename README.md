# Disgaea DS Manager - Flathub Submission

### Please confirm your submission meets all the criteria

-   [x] Please describe the application briefly.

**Disgaea DS Manager** is an editor for DSARC and MSND archives, built with C# (.NET 10.0). It provides a simple Avalonia UI GUI interface to create and edit archives for Disgaea DS modding.

**Key Features:**

-   Edit DSARC and MSND archive formats (DSARC IDX coming soon with DSARC being renamed to it's proper DSARC FL)
-   Intuitive Avalonia UI interface
-   Support for both x86_64 and ARM64 architectures
