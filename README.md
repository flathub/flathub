# TrueConf Client

This repo hosts the flatpak wrapper for the desktop version of [TrueConf Client](https://trueconf.com/) and does not contain any TrueConf-licensed binary code.
Instead, it is downloaded and converted to flatpak format on the fly from the official TrueConf repository by running the flatpak install/update commands.

**NOTICE: This wrapper is not verified by, affiliated with, or supported by TrueConf.**

## Issue reporting

**Please only report issues in this repo that are specific to the flatpak wrapper.**

Issues that can be replicated in another release should be reported via [TrueConf's support page](https://trueconf.com/support/).
Make sure, that the reported issue is **not** flatpak-related.
