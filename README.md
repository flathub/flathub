# OneDrive Client for Linux

## Usage
```
flatpak run io.github.abraunegg.onedrive <options>
```
Add an alias that allows you to directly call ```onedrive```:
```
echo "alias onedrive='flatpak run io.github.abraunegg.onedrive'" >> ~/.bashrc
```
