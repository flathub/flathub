# Postman with Git (Flatpak app)

Postman requires Git for its new [Native Git](https://learning.postman.com/docs/agent-mode/native-git/) feature.

This fork was created because the original [com.getpostman.Postman Flatpak app](https://github.com/flathub/com.getpostman.Postman)
does not include Git and does not allow Postman to use Git from the host OS.
Please upvote [GitHub issue](https://github.com/flathub/com.getpostman.Postman/issues/466)
and [GitHub PR](https://github.com/flathub/com.getpostman.Postman/pull/472).

Install: `flatpak install io.github.yaal.PostmanGit`
